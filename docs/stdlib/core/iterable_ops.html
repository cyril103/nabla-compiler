IterableFactory

trait IterableFactory[CC[_]]

Factory generique pour une famille de collections a un parametre de type.

CC[_] represente un constructeur de type d'arite 1, par exemple List ou Set. La factory permet aux operations polymorphes de reconstruire une collection de la meme famille sans coder directement la collection concrete.

IterableOps

trait IterableOps[A, CC[_], C]

Base experimentale pour des operations de collection a la Scala IterableOps.

Cette version expose le lien entre l'Iterable courant, sa famille CC[_] et le type concret C. Les methodes map/filter sont encore experimentales et reconstruisent leur resultat via IterableFactory.newBuilder.
