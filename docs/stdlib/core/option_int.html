Option Int

import core.option_int

Variante specialisee de Option[Int]. Elle evite de fournir une valeur par defaut lors de la creation d'une option vide pour les entiers. Preferer Option[Int] dans le code generique; garder OptionInt quand cette representation specialisee rend le code plus direct.
