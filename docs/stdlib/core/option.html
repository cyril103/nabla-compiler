Option

Valeur optionnelle generique. Option[T] represente soit une valeur presente, soit une absence. C'est le type a privilegier quand une operation peut ne rien produire sans que ce soit une erreur. Utiliser Option.some[T](value) pour construire une valeur presente et Option.none[T]() pour construire une absence. Les anciens noms optionSome[T](value), Option.none[T](default) et optionNone[T](default) restent disponibles pour compatibilite.

Option

class Option[T](defined: Bool, value: T)

Conteneur optionnel generique.

Les methodes principales sont isDefined, isEmpty, nonEmpty, map, flatMap, foreach, filter, orElse et getOrElse.
