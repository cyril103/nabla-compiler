Module

Util

import util

Generateur pseudo-aleatoire deterministe. Les fonctions retournent la valeur generee avec le nouvel etat. Conserver et reutiliser cet etat permet de produire une sequence reproductible; utiliser randomSeedNow() quand une seed temporelle suffit.
