import io

IO

Fonctions d'entree-sortie simples. Les operations de fichier retournent un `Bool` ou un code `Int` pour laisser le programme decider comment reagir aux erreurs. Les fonctions de ce module sont des wrappers utilisateur autour des primitives runtime bas niveau.

def println(value: String): Unit

Affiche une chaine suivie d'un retour a la ligne.

def Recommandee

readTextFile

def readTextFile(path: String): String

Lit tout le contenu d'un fichier texte.

Si le fichier ne peut pas etre lu, la primitive runtime retourne une chaine vide.

def Recommandee

writeTextFile

def writeTextFile(path: String, content: String): Int

Ecrit `content` dans un fichier texte en remplacant son contenu.

Retourne le nombre de bytes ecrits, ou `-1` en cas d'erreur runtime.

def Recommandee

appendTextFile

def appendTextFile(path: String, content: String): Int

Ajoute `content` a la fin d'un fichier texte.

Retourne le nombre de bytes ecrits, ou `-1` en cas d'erreur runtime.

def Recommandee

renameTextFile

def renameTextFile(from: String, to: String): Bool

Renomme ou deplace un fichier texte.

Retourne `true` si l'operation reussit, `false` sinon.

def Recommandee

def Recommandee

def deleteTextFile(path: String): Bool
