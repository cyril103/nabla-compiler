Module

IO

import io

Fonctions d'entree-sortie simples. Les operations de fichier retournent un Bool ou un code Int pour laisser le programme decider comment reagir aux erreurs. Les fonctions de ce module sont des wrappers utilisateur autour des primitives runtime bas niveau.
