Map

Dictionnaires immuables cle-valeur. Utiliser Map("name" -> value) pour creer une map depuis des paires, Map.empty[K, V]() pour construire progressivement une map, ou Map.fromArray[K, V](entries) pour construire depuis un tableau de paires.

Map

class Map[K, V]

Dictionnaire immutable.

Les operations updated, removed et clear retournent une nouvelle map. Si plusieurs paires utilisent la meme cle, la derniere valeur gagne.
