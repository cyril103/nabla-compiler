Module

Map

import collections.map

Dictionnaires immuables cle-valeur. Utiliser Map("name" -> value) pour creer une map depuis des paires, Map.empty[K, V]() pour construire progressivement une map, ou Map.fromArray[K, V](entries) pour construire depuis un tableau de paires. Les tableaux generiques visibles dans certains retours sont une representation actuelle ; l'API recommandee reste Map et Array[T]. Les helpers map... et les ObjectArray internes ne sont pas une surface applicative idiomatique.

class MapBuilder[K, V] extends Builder[Tuple2[K, V], Map[K, V]]

Builder experimental pour produire une Map[K, V] depuis des paires.

Les ajouts successifs appliquent la semantique de Map.fromArray: si une cle est ajoutee plusieurs fois, la derniere valeur gagne dans le resultat.
