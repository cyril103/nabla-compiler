Module

Buffer

import collections.buffer

Traits publics pour les sequences mutables a taille variable. Cette surface s'inspire de scala.collection.mutable.Buffer: un Buffer est une collection parcourable qui peut grandir, retrecir et modifier ses elements par index. Nabla garde des noms explicites et retourne Unit pour les primitives de mutation; les methodes *InPlace retournent le buffer pour permettre le chainage polymorphe.

trait Experimental

Buffer

trait Buffer[A] extends Iterable[A]

Sequence mutable a taille variable.

Les implementations fournissent les operations indexees primitives get/set/insert/removeAt/removeRange. Les helpers append, prepend, trimStart, dropInPlace, takeInPlace, sliceInPlace, subtractOne, filterInPlace et padToInPlace sont derives de ces primitives, comme dans la hierarchie Scala mutable.Buffer.
