trait Recommandee

Iterator

trait Iterator[T]

Curseur mutable avance explicitement sur une sequence d'elements.

Un Iterator[T] represente un parcours a usage sequentiel: hasNext() indique si next() peut encore produire une valeur, puis next() avance le curseur.
