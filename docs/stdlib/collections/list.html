Module

List

import collections.list

Liste chainee immutable experimentale. List[T] est inspiree de Scala. La liste vide est le singleton runtime `Nil`, declare comme `List[Nothing]` et assignable a `List[T]`. Utiliser `List.empty[T]()` et `List.cons[T](head, tail)` pour construire les listes.

object Recommandee

Nil

object Nil extends List[Nothing]

Liste vide singleton, Scala-like.

`Nil` est un objet runtime qui implemente `List[Nothing]`. Comme `Nothing` est le type bottom, `Nil` peut etre utilise partout ou une `List[T]` est attendue.
