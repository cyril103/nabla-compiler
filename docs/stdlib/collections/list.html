List

import collections.list

Liste chainee immutable experimentale. List[T] est inspiree de Scala, mais la V0 utilise encore Nil[T](defaultValue) au lieu d'un singleton Nil polymorphe. Utiliser List.empty[T](defaultValue) et List.cons[T](head, tail) pour construire les listes.
