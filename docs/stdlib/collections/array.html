Modules / collections.array

import collections.array

Array

Tableaux generiques pour le code utilisateur. Importer collections.array donne acces a Array[T], Array.empty[T](), Array.fill[T], Array.tabulate[T] et Array.range / Array.rangeUntil. Les anciens noms ArrayFill[T] et ArrayRange restent disponibles pour compatibilite. Les types bas niveau comme IntArray, ArrayInt, ObjectArray[T] et ArrayObject[T] restent des details de stdlib dans les cas courants.

class Recommandee

Array

class Array[T](size: Int)

Tableau mutable de taille fixe.

Array[T] est le nom public a utiliser. Le compilateur choisit la representation adaptee selon T.

def Compatibilite

ArrayFill

def ArrayFill[T](size: Int, value: T): Array[T]

Cree un tableau rempli avec la meme valeur.

Preferer Array.fill[T](size, value) dans le nouveau code.

def Compatibilite

ArrayRange

Cree un tableau d'entiers contenant 0, 1, ..., size - 1.

Preferer Array.range(size) dans le nouveau code.

def Recommandee

fill

def Array.fill[T](size: Int, value: T): Array[T]

Cree un tableau rempli avec la meme valeur.

Pour T = Int, Long, Float, Double ou Bool, le compilateur redirige ce nom vers la specialisation primitive adaptee. Pour les autres types, Array[T] est represente par ArrayObject[T].
