import collections.set

Set

Ensembles immuables avec valeurs uniques. Utiliser Set(values...) pour creer un ensemble depuis des elements, Set.empty[T]() pour construire progressivement un ensemble, ou Set.fromArray[T](values) pour dedupliquer un tableau. Les anciens noms SetEmpty[T]() et SetFromArray[T](values) restent disponibles pour compatibilite, mais ne sont plus l'API recommandee pour du code applicatif.

fromArray

def Set.fromArray[T](values: Array[T]): Set[T]

Deduplique un tableau en ensemble.

Pour Array[Int], Array[Long], Array[Float], Array[Double] et Array[Bool], le compilateur redirige ce nom vers la specialisation adaptee. Pour les autres types, Array[T] est represente par ArrayObject[T].

class Recommandee

Set

class Set[T]

Ensemble immutable.

Les operations comme add, remove, union, intersect et difference retournent un nouvel ensemble. L'instance d'origine n'est pas modifiee. L'egalite repose sur == et l'index interne utilise hashCode(), ce qui permet aux overrides utilisateur de participer a la deduplication.
