Module

Set

import collections.set

Ensembles immuables avec valeurs uniques. Utiliser Set(values...) pour creer un ensemble depuis des elements, Set.empty[T]() pour construire progressivement un ensemble, ou Set.fromArray[T](values) pour dedupliquer un tableau. toArray() retourne encore la representation generique actuelle ; preferer Array[T] pour construire ou passer de nouveaux tableaux.

class Recommandee

Set

class Set[T] extends IterableOps[T, Set, Set[T]]

Ensemble immutable.

Les operations comme add, remove, union, intersect et difference retournent un nouvel ensemble. L'instance d'origine n'est pas modifiee. L'egalite repose sur == et l'index interne utilise hashCode(), ce qui permet aux overrides utilisateur de participer a la deduplication. Les operations generiques heritees d'IterableOps, comme map et filter, reconstruisent un Set via SetFactory et conservent donc la deduplication.
