Modules / collections.set

import collections.set

Set

Ensembles immuables avec valeurs uniques. Utiliser `SetEmpty[T]()` pour construire progressivement un ensemble, ou `SetFromArray[T](values)` pour dedupliquer un tableau. Les fonctions commencant par `set` restent disponibles pour compatibilite, mais ne sont pas l'API recommandee pour du code applicatif.

def

SetFromArray

def SetFromArray[T](values: Array[T]): Set[T]

Deduplique un tableau en ensemble.

Pour `Array[Int]`, `Array[Long]`, `Array[Float]`, `Array[Double]` et `Array[Bool]`, le compilateur redirige ce nom vers la specialisation adaptee. Pour les autres types, `Array[T]` est represente par `ArrayObject[T]`.

def

SetEmpty

def SetEmpty[T](): Set[T]

Cree un ensemble vide.

class

Set

class Set[T]

Ensemble immutable.

Les operations comme `add`, `remove`, `union`, `intersect` et `difference` retournent un nouvel ensemble. L'instance d'origine n'est pas modifiee.
