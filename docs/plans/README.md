`v0.1.0`; leur résultat est résumé dans `AGENTS.md`, `docs/roadmap.md` et les
notes de release `docs/releases/0.1.md`.

post-0.1 courant est décrit dans `docs/roadmap.md` : durcissement héritage/runtime,
nettoyage de la surface stdlib et maintien de `docs/internals.md`.
